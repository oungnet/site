# Ban Non-Yai Smarter! Platform

บูรณาการนวัตกรรมเพื่อความเท่าเทียม สนับสนุนผู้พิการ ผู้ประกอบการ และชุมชน

## 📁 โครงสร้างไฟล์

```
site/
├── client/
│   └── src/
│       ├── pages/           # หน้าต่างๆ ของเว็บไซต์
│       ├── components/      # คอมโพเนนต์ React
│       ├── App.tsx          # ไฟล์หลักของแอปพลิเคชัน
│       └── index.css        # สไตล์ CSS
├── skills/
│   └── techgreen-website-builder/  # ทักษะสำหรับสร้างเว็บไซต์
│       ├── SKILL.md         # คู่มือการสร้างเว็บไซต์
│       ├── references/      # เอกสารอ้างอิง
│       ├── scripts/         # สคริปต์ช่วยเหลือ
│       └── templates/       # เทมเพลต
└── README.md               # ไฟล์นี้

```

## 🚀 หน้าต่างๆ

1. **Home** - หน้าแรกของเว็บไซต์
2. **DisabilityBenefits** - ข้อมูลสิทธิประโยชน์ผู้พิการ
3. **TaxBenefits** - ข้อมูลสิทธิลดหย่อนภาษี
4. **Resources** - ข้อมูลทรัพยากรและที่ดินราชพัสดุ
5. **Innovation** - ข้อมูลนวัตกรรมเกษตรสีเขียว
6. **Partnership** - ข้อมูลความร่วมมือทางธุรกิจ
7. **Dashboard** - แดชบอร์ดและสถิติ
8. **Learning** - ศูนย์การเรียนรู้และบทความ
9. **Register** - หน้าลงทะเบียน
10. **Login** - หน้าเข้าสู่ระบบ
11. **ApplyBenefits** - แบบฟอร์มสมัครสิทธิประโยชน์
12. **Contact** - หน้าติดต่อเรา
13. **UserDashboard** - แดชบอร์ดส่วนตัวของผู้ใช้

## 🎨 ทักษะ (Skill)

ทักษะ **techgreen-website-builder** ให้คู่มือการสร้างเว็บไซต์ที่คล้ายกัน พร้อมรูปแบบคอมโพเนนต์ แบบฟอร์ม และคู่มือการจัดแต่งรูปแบบ

## 📝 เนื้อหา

- ข้อมูลสิทธิประโยชน์ผู้พิการปี 2568
- สิทธิลดหย่อนภาษี (มาตรา 33, 35, ESG Tax Credit)
- ข้อมูลที่ดินราชพัสดุและระบบจอง
- นวัตกรรมเกษตรสีเขียว
- ความร่วมมือทางธุรกิจ
- แดชบอร์ดและสถิติ
- บทความและแหล่งเรียนรู้

## 🛠️ เทคโนโลยี

- React 19
- Tailwind CSS 4
- shadcn/ui
- TypeScript
- Wouter (Routing)

## 📞 ติดต่อ

สำหรับข้อมูลเพิ่มเติม โปรดเยี่ยมชมเว็บไซต์หรือติดต่อสำนักงาน
